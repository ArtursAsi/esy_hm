### Instructions:
Clone project
>git clone https://github.com/ArtursAsi/easy_hm.git

Install Composer
>composer install

Install npm
>npm install

Create .env file
>cp .env.example .env

Create database
>link in .env file

Migrate Database
>php artisan migrate

Generate key
>php artisan key:generate

Run app 
>php artisan serve
